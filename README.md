## Instruction

1. Clone the repository:

2. Open the cloned repository in the code editor.

  - If you are using Visual Studio Code as code editor, try `Ctrl + k Ctrl + 0 ` or `Cmd + k Cmd + 0` shortcut to collapse all file.
